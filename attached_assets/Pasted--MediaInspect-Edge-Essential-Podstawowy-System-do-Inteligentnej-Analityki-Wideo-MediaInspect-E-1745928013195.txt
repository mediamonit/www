# MediaInspect Edge Essential

## Podstawowy System do Inteligentnej Analityki Wideo

MediaInspect Edge Essential to kompaktowe rozwiązanie łączące oprogramowanie do analizy wideo z niezależnym źródłem zasilania, zaprojektowane z myślą o prostych zastosowaniach monitoringu wymagających lokalnej analityki w terenie.

![MediaInspect Edge Essential](images/essential-product.jpg)

## Zastosowania

- **Tymczasowy monitoring placów budowy**
- **Zabezpieczenie odległych obiektów**
- **Monitoring środowiskowy**
- **Autonomiczne punkty obserwacyjne**
- **Mobilne stanowiska ochrony**

## Specyfikacja Techniczna

### Oprogramowanie MediaInspect Edge Essential

| Funkcjonalność | Specyfikacja |
|----------------|--------------|
| Obsługiwane strumienie | 1 strumień wideo |
| Rozdzielczość | Do 1080p (Full HD) |
| Algorytmy analityczne | Detekcja ruchu, przekroczenie linii, rozpoznawanie podstawowych obiektów |
| Latencja | <5ms |
| Buforowanie lokalne | Do 128GB (7 dni nagrań) |
| Zaawansowane funkcje | Harmonogramy aktywności, strefy zainteresowania, filtry obrazu |
| Interfejs zarządzania | Przeglądarkowy, API REST, aplikacja mobilna |
| Aktualizacje | OTA (over-the-air) |
